# 凯撒密码加密/解密工具

一个基于 Python 实现的简易凯撒密码工具，专注于固定偏移量（ASCII 码 +3 加密、-3 解密）的文本转换，适合基础学习与日常简单加密需求。

## 三、基础知识点
    1、变量与空字符串初始化， result = ""
        - result 是一个变量，用来存储最终加密 / 解密后的字符串
        - "" 表示空字符串，用于逐步拼接字符
    
    2、for 循环遍历字符串， for char in text:
        - 逐个字符处理，实现逐个加密 / 解密。

    3、内置函数 ord () —— 获取字符 ASCII 码
       内置函数 chr () —— ASCII 码转回字符

    4、字符串拼接， result += chr(new_ascii)
        - 等价于：result = result + chr(new_ascii)
        - 把每个转换后的字符拼接到最终结果里

    5、数学运算， new_ascii = ascii_code + 3   new_ascii = ascii_code - 3
        - 使用 加法 + 和 减法 - 实现偏移
        - 这是 Python 最基础的算术运算语法
    
    6、函数定义语法，  def caesar_encrypt(text):
        - 使用 def 关键字 定义函数
        - caesar_encrypt 是函数名（见名知意）
        - (text) 是函数的参数，接收外部传入的文本
        - 冒号 : 表示代码块开始
        - 函数内部实现加密逻辑，最后通过 return 返回结果

    7、主程序入口（程序运行起点）  if __name__ == "__main__":
        -Python 标准主入口语法

    8、if /elif/else 条件判断   
        if mode == "加密":
            ...
        elif mode == "解密":
            ...
        else:
            ...
        - 根据用户选择，执行加密或解密。

    9、print () 输出函数    print("加密结果：", final)
        - print() 是控制台输出语法，用于在屏幕上显示文字、结果
